You can change keys while s key is pressed down. Simple Vi Mode will be active by the following procedure.

  1. s key_down
  2. any keys of vi operation key_down (h, j, k, l,...)
  3. any keys of vi operation key_up (h, j, k, l,...)
  4. Then Simple Vi Mode is active until you release s key.
Example:
  • Press s
  • Press j
  • Press k
  • Release j
  • Release k
  • Release s
Key bindings:
  • h: left arrow
  • j: down arrow
  • k: up arrow
  • l: right arrow
  • f: fn
  • c: left control
  • v: left option
  • b: left command
