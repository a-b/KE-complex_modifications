Caps Lock to Debug in Various IDE

Use the same keyboard shortcuts to debug across various IDEs. Hold the Caps Lock key and press the other key to perform the action.

Mapping

Key Maps To
⇪ Caps Lock + ↓ Down Arrow Step Over
⇪ Caps Lock + ← Left Arrow Step Out
⇪ Caps Lock + → Right Arrow Step In
⇪ Caps Lock + ↵ Enter Continue

Supported IDEs

  • Xcode
  • Intellij
  • Visual Studio Code (VSCode)
