Keychron K6 for es-LATAM users

This profile allows you to insert accented characters using [ + key combinations. This configuration facilitates accenting for users accustomed to ISO Latam distributions.

Key Combinations

Key Combination Outputs
a, e, i, o, u [ + vowel á, é, í, ó, ú
A, E, I, O, U [ + shift + vowel Á, É, Í, Ó, Ú
n [ + n ñ
N [ + shift + n Ñ
